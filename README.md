### IP Geolocation Tracker Tool From BlackHatCommunity

![alt text](https://github.com/BlackHatLay/geolocation/blob/main/BHC.jpg?raw=true)
### Installation (Termux)
  ```
  cd $HOME
  ```
  ```
  pkg install git -y
  ```
  ```
  git clone https://github.com/FSocietyMM/geolocation
  ```
  ```
  cd geolocation
  ```
  ```
  bash setup.sh
  ```
### Installation (Kali Terminator)
  ```

  ```
